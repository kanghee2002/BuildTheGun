## 아이디어 메모 (자유롭게 적기)

### 사용 규칙
- 이 문서는 **떠오르는 아이디어를 검열 없이 빠르게 적는 곳**입니다.
- 문장/형식이 거칠어도 괜찮습니다. 대신, 나중에 실행 가능하도록 최소한의 맥락만 남깁니다.
  - 기대 효과(왜 좋은지)
  - 적용 화면/상황(언제 보이는지)
  - 구현 난이도/리스크(알면)

---

### UI/UX 아이디어

#### 합체 카드 비주얼/인터랙션
- 합체 카드는 카드 테두리가 겹쳐 보이는 **“레이어드(Layered)” 비주얼**로 표시합니다.
  - 2장 합체: 2겹
  - 3장 합체: 3겹
- **빌드 편집 화면 / 일시정지 화면**처럼 플레이어가 꼼꼼히 볼 수 있는 순간에는,
  - 합체 카드를 **탭(모바일 대응) 또는 호버(PC)** 하면,
  - 개별 구성 카드가 **펼쳐져 보이도록** 표시합니다.

### 게임플레이/카드 아이디어

#### 강화 총알
- 이전 총알이 연발된 만큼, 다음 카드의 연발 수를 추가로 증가

#### 빌드: 연발 빌드
- 연발 수를 지속적으로 키우는 빌드 컨셉
