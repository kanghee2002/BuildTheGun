## 기능 목록 (Features)

### 1. 개요
- 이 문서는 게임에 포함될 **주요 기능 목록과 상태**를 관리합니다.
- 각 기능은 “기획 → 확정 → 개발중 → 완료” 흐름을 가집니다.

### 2. 기능 상태 레이블
- **IDEA**: 아이디어 단계, 아직 실험/검토 필요
- **PLANNED**: 구현 확정, 우선순위/범위 정해짐
- **IN_PROGRESS**: 개발 중
- **DONE**: 개발 및 기본 QA 완료
- **DEPRECATED**: 더 이상 사용하지 않거나 삭제 예정

### 3. 기능 테이블

| ID | 이름 | 설명 | 상태 | 우선순위(High/Mid/Low) | 비고 |
|----|------|------|------|------------------------|------|
| F-001 | 코어 전투 시스템 | 이동/사격/피격/사망까지의 기본 전투 플로우 | PLANNED | High | |
| F-002 | 무기 커스터마이징 | 총기 파츠 장착/해제, 스텟 변화 | IDEA | High | |
| F-003 | 인벤토리 UI | 아이템 보기/정렬/상세 보기 | IDEA | Mid | |

> 새로운 기능을 만들 때마다 **줄을 추가**하고, 상태를 업데이트하면서 관리합니다.

### 4. 로드맵(선택)
- **MVP(최초 플레이 가능 버전)**:
  - 예: 전투 시스템, 1종의 무기, 1개 맵, 기본 HUD
- **버전 1.0**:
  - 예: 3종 무기, 3개 맵, 튜토리얼, 기본 세이브/로드
- **이후 업데이트 후보**:
  - 예: 코옵 모드, 시즌제 컨텐츠, 랭킹 시스템
